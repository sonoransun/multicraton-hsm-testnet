# Craton HSM Sponsors

Craton HSM is built by [Craton Software Company](https://github.com/craton-co), a small team working to bring memory-safe, post-quantum-ready cryptographic infrastructure to everyone. Your sponsorship directly funds development, security audits, and community support.

## Why Sponsor?

Craton HSM replaces aging C++ HSM software with a modern Rust implementation that eliminates entire classes of memory-safety vulnerabilities. We're also one of the first PKCS#11 implementations to ship post-quantum cryptography (ML-KEM, ML-DSA, SLH-DSA), preparing the ecosystem for NIST's PQC migration timeline.

Every dollar goes toward:
- Continued development and maintenance
- Security audits and fuzzing infrastructure
- CI/CD across Linux, macOS, and Windows
- Documentation and integration guides
- Working toward FIPS 140-3 certification
